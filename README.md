# ❤️ DatingApp

A modern Full Stack Dating Application built using ASP.NET Core Web API and Angular.

The application demonstrates enterprise-level backend and frontend development practices including authentication, authorization, profile management, messaging, photo management, filtering, pagination, middleware, and scalable architecture.
